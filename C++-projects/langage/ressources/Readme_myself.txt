Faire un programme (en bash) qui permet d'interroger la base de donnees et de recuperer des mots avec les infos necessaires, un par un.
L'utilisateur y ajoutera les donnees manquantes (registres, synonyme...) et ce sera enregistre dans une nouvelle base de donnees personnelle pour utiliser en local.
Le programme devra checker s'il n'y a pas des doublons, chercher au hasard ou avec critere. Et chaque mot rentre dans la nouvelle base doit etre supprimee de l'ancienne.
Un bon petit challenge pour apprendre le bash! Et construire sa propre base de donnees!
