Le but est de faire un script qui lit les entrees dans librairie.txt et les ajoute dans mylibrairie.txt
Chaque entree lue doit etre supprimee de l'ancienne base et ecrite dans la nouvelle
On demandera a l'utilisateur d ajouter des champs a l'input pour etre standardisee dans l'output

Pour l'exemple, la librairie.txt contient des mots, l'utilisateur construira une base de donnees qui demande d indiquer si le mot est verbe,nom commun,adjectif et d'ajouter des synonymes(qui contiendra seulement a terme le lemme et qu'on ira lire comme une nouvelle entree dans la base (pour cela il doit exister dans la base en tant qu'entree))
